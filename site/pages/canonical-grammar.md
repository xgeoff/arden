## Canonical Arden Grammar

This page defines the canonical grammar used across the Arden guide.

## Core Clause Structure

Arden normally uses **subject-verb-object** order.

| English | Arden |
| --- | --- |
| I read the book. | **Me leraa levran.** |
| You see the house. | **Tu visaa domn.** |
| We meet later. | **Mena miraa soren.** |

## Verb System

Arden verbs use a suffix-tense system.

| Tense | Form | Example with `vare` |
| --- | --- | --- |
| present | root form | **me vare** |
| past | `-o` | **me varo** |
| future | `-bo` | **me varbo** |

Verbs do **not** agree with person or number.

Examples:

| English | Arden |
| --- | --- |
| I go. | **Me vare.** |
| I went. | **Me varo.** |
| I will go. | **Me varbo.** |
| They read. | **Sena leraa.** |
| They read earlier. | **Sena lerao.** |
| They will read. | **Sena leraabo.** |

## Copula

Arden has an explicit copula: **`es`**.

Use `es` when you want a clear linking verb for:

- identity
- predication
- equational statements
- clauses where the relation would otherwise feel unclear

Examples:

| English | Arden |
| --- | --- |
| My name is Kareth. | **Melun namor es Kareth.** |
| The tree is green. | **Ten silvar es livenel.** |
| The weather is wonderful. | **Solir es miravel.** |

### Zero Copula

Zero copula is also valid, especially in:

- casual speech
- short descriptive clauses
- headlines or fragments
- highly predictable predicates

Examples:

| English | Arden |
| --- | --- |
| I am busy today. | **Me prethel solinor.** |
| Everything is okay. | **Orel olven.** |
| Miserable night. | **Grelvel nareth.** |

### Tense With The Copula

The copula follows the same tense logic:

| Tense | Form |
| --- | --- |
| present | **es** |
| past | **eso** |
| future | **esbo** |

Examples:

| English | Arden |
| --- | --- |
| It is good. | **Es melor.** |
| It was good. | **Eso melor.** |
| It will be good. | **Esbo melor.** |

## Questions

Arden does not invert word order.

| Type | Pattern | Example |
| --- | --- | --- |
| yes / no | statement + `kaa` | **Tu lirel, kaa?** |
| open question | `ke` + clause | **Ke tu mentaa?** |
| suggestion prompt | `ken` + topic/clause | **Ken kareth?** |

## Negation

Use `no` before the verb or predicate.

| English | Arden |
| --- | --- |
| I do not agree. | **Me no agree.** |
| I did not go. | **Me no varo.** |
| You are not late. | **Tu no es tarel.** |
| You are not late. | **Tu no tarel.** |

Both copular and zero-copula negation are acceptable when the meaning is clear.

## Aspect And Time Nuance

Tense is carried by the verb. Aspect particles are optional refinements.

| Particle | Meaning |
| --- | --- |
| `rin` | ongoing / currently |
| `sen` | habitual / these days |
| `jurel` | just / recently |
| `pren` | about to / imminent |

Examples:

| English | Arden |
| --- | --- |
| I am going right now. | **Me vare rin.** |
| I just went. | **Me varo jurel.** |
| You work these days. | **Tu labore sen.** |
| I am about to go. | **Me vare pren.** |

## Modals

Modals come before the main verb.

| Modal | Meaning |
| --- | --- |
| `kan` | can / could |
| `shal` | should |
| `wud` | would |
| `meliv` | might |

Examples:

| English | Arden |
| --- | --- |
| I can help. | **Me kan helpe.** |
| You should go. | **Tu shal vare.** |
| We would stay. | **Mena wud stare.** |

## Nouns And Adjectives

Many lexical tables preserve traditional inflected forms:

| Function | Ending |
| --- | --- |
| nominative | `-a` |
| accusative | `-am` |
| genitive | `-um` |
| plural | `-ee` |

Adjectives normally follow the noun and commonly end in `-el`.

| English | Arden |
| --- | --- |
| quiet night | **nareth humel** |
| safe road | **meren surel** |
| green tree | **silvar livenel** |

## Possession

Use `de` for ordinary possession and association.

| English | Arden |
| --- | --- |
| the teacher's book | **Levrana de lertha.** |
| the light of the sun | **Lurena de solira.** |

`yun` may still be used as a poetic or elevated alternative.
