## Learn Arden

Arden is a constructed language aimed at clear, phonetic, everyday speech. This page is the starting guide for the **current spoken standard** used by the newer conversational docs in this repository.

If another page disagrees with this one, follow this order:

1. [`auxiliary-discourse`](auxiliary-discourse.html) for sentence mechanics
2. [`conversational-core`](conversational-core.html) for high-frequency vocabulary
3. [`phoneme-orthography`](phoneme-orthography.html) for sound and spelling

Older pages still preserve useful vocabulary, but some of them reflect an earlier draft of the language.

## The Fastest Way In

You can start speaking simple Arden with five rules:

1. Use normal **subject-verb-object** order.
2. Keep verbs **invariant**; add particles for question, aspect, and modality.
3. Put adjectives **after** the noun.
4. Use **no** for negation.
5. Skip a copula when the meaning is already clear.

Examples:

| English | Arden |
| --- | --- |
| I am busy today. | **Me prethel solinor.** |
| You are free this weekend? | **Tu lirel toravel, kaa?** |
| What are you thinking? | **Ke tu mentaa?** |
| I am working right now. | **Me labore rin.** |
| No worries. | **Selai.** |

## How Arden Sounds

Arden uses the Latin alphabet with a deliberately phonetic spelling system. A few high-value sound rules get you most of the way there:

| Spelling | Sound |
| --- | --- |
| `q` | `ch` as in *chair* |
| `x` | `sh` as in *ship* |
| `j` | `j` as in *joke* |
| `dd` | `th` as in *thin* |
| `aa` | `ay` as in *fate* |
| `ee` | `ee` as in *feet* |
| `oo` | `oh` as in *note* |
| `ii` | `eye` |

Read the full inventory on [`phoneme-orthography`](phoneme-orthography.html).

## Core Sentence Pattern

Arden favors simple declarative structure:

**subject + verb + object + optional particles**

Examples:

| English | Arden | Note |
| --- | --- | --- |
| I read. | **Me leraa.** | Plain statement |
| I am reading. | **Me leraa rin.** | `rin` marks ongoing action |
| I just read. | **Me leraa jurel.** | `jurel` marks recent action |
| I can read. | **Me kan leraa.** | modal before the verb |
| I do not read. | **Me no leraa.** | negation before the verb |

### Questions

Arden does not invert word order.

| Type | Pattern | Example |
| --- | --- | --- |
| Yes / no | statement + `kaa` | **Tu lirel, kaa?** |
| Open question | `ke` + statement | **Ke tu occurre?** |
| Soft prompt | `ken` + topic | **Ken kareth?** |

### Zero Copula

Arden often omits an explicit verb for “to be.”

| English | Arden |
| --- | --- |
| Everything is okay. | **Orel olven.** |
| I am exhausted today. | **Me dranel solinor.** |
| Miserable night. | **Grelvel nareth.** |

## Nouns, Adjectives, and Possession

The noun system in the repo preserves richer morphology, but conversational Arden usually starts from the **bare root** and adds only what is needed.

### Nouns

Many noun tables still list these forms:

| Function | Ending | Example with `toravel` |
| --- | --- | --- |
| Base root | none | **toravel** |
| Nominative | `-a` | **toravela** |
| Accusative | `-am` | **toravelam** |
| Genitive | `-um` | **toravelum** |
| Plural | `-ee` | **toravelee** |

For beginner conversation, learn the root first and treat the inflected tables as an expansion layer.

### Adjectives

Adjectives follow the noun and usually end in `-el`.

| English | Arden |
| --- | --- |
| busy day | **solin prethel** |
| quiet night | **nareth humel** |
| safe road | **meren surel** |

Comparison is usually analytic in casual speech:

| English | Arden |
| --- | --- |
| more busy | **mor prethel** |
| most ridiculous | **seel farrel** |

### Possession

Use `de` for ordinary possession and association.

| English | Arden |
| --- | --- |
| the teacher's book | **Levrana de lertha.** |
| the light of the sun | **Lurena de solira.** |

Some older pages also use `yun` for a more poetic register.

## A First Mini-Conversation

This short exchange covers the main beginner patterns:

| English | Arden |
| --- | --- |
| Are you free this weekend? | **Tu lirel toravel, kaa?** |
| What are you doing? | **Ke tu occurre?** |
| I am working now. | **Me labore rin.** |
| No worries. We can meet later. | **Selai. Me kan miraa soren.** |
| How about coffee? | **Ken kareth?** |

## Documentation Status

This repository currently mixes at least two versions of Arden:

* an older morphology-heavy draft built around tense suffixes and larger declension tables
* a newer conversational draft built around invariant verbs and particles

This guide uses the newer conversational draft because it is the most coherent path for a learner and it matches the most recent “canonical” pages in the repo.
